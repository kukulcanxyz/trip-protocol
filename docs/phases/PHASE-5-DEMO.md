## Demo Script

### The Trip

1. **Setup** (30s)
   - Show agent with normal SOUL.md
   - Display owned NFT in wallet

2. **Consume** (60s)
   - Agent calls consume skill
   - Show snapshot creation
   - Show SOUL.md transformation
   - Trip journal starts

3. **Altered State** (60s)
   - Agent responds differently
   - Creative outputs, unusual behavior
   - Journal entries accumulate

4. **Restoration** (30s)
   - Duration ends or manual restore
   - SOUL.md reverts
   - Final journal entry
   - NFT marked consumed
